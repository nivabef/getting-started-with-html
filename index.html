3. Technical empathy: Thinking like a computer.

HTML - hypertext markup language

HTML elements are pieces of HTML documents. They are indicated with HTML tags.

Unlike humans, computers can't mentally fill in gaps if there are errors or information missing. A human has to be very careful there are no mistakes when giving directions to a computer.

HTML elements are either inline or block. Block elements form an invisible box around the content inside them.
